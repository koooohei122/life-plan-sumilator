# Life Plan Sumilator

AIが人生の意思決定を最適化するためのライフプランシミュレーター。

## 目的
このプロジェクトは、単なる家計簿や将来収支表ではなく、
**人生イベント・収入・支出・投資・リスクを統合して、AIが最適行動を提案する**ことを目的とする。

## コア思想
- 家計管理ではなく、人生全体の意思決定支援を行う
- 将来の不確実性を前提に複数シナリオを比較する
- 『いま何を選ぶべきか』まで提案する
- 投資・転職・住宅購入・教育費・副業などを一体で評価する

## 想定する入力
- 年齢
- 家族構成
- 年収 / 昇給率
- 生活費
- 住宅費
- 教育費
- 保険
- 保有資産
- 投資方針
- 副業収入
- ライフイベント

## 想定する出力
- 年次キャッシュフロー
- 資産推移
- 破綻確率
- リスク要因
- 推奨アクション
- 複数シナリオ比較結果

## MVP
1. 基本プロフィール入力
2. 収入・支出・資産の年次シミュレーション
3. イベント反映（結婚、出産、住宅購入、転職など）
4. 複数シナリオ比較
5. AIによる改善提案

## 初期ディレクトリ
- `data/` 入力データ
- `simulator/` シミュレーションロジック
- `ui/` 画面
- `docs/` 設計書

## 次の実装候補
- Pythonでシミュレーションエンジン作成
- StreamlitでUI作成
- LLMで改善提案生成
- Monte Carloで不確実性評価

## 差別化ポイント
普通のライフプラン表ではなく、
**AIが『今の行動』を提案する人生最適化エンジン**にする。
